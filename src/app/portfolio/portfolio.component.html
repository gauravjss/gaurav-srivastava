Pharmacy Locator

Environment: Angular 4, Node JS, HTML5, CSS3.

Client: CVS Caremark
Project Title: Pharmacy Locator
Role: Technical Lead
Description: The purpose of this project is to create a new stand along FAST component to support request new RX functionality. The application has ability to login to the CVS Caremark website, check for the drugs, request new prescription, search for an alternatives. User is having an option to set the default pharmacy, change the default pharmacy, search for a drug at the selected pharmacy and gets the price of the drug. He can check how much his plan will pay and how much he has to pay.
